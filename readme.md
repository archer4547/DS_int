Solution contains 2 main files: "MnistClassifiers.py" and "examples.ipynb", and an additional file that helps with installing the dependencies called "requirements.txt".

"MnistClassifiers.py" has the classes for FFNN, CNN and RF classifiers, and also the MnistClassifier class that containes them all.

"examples.ipynb" is an example notebook where you can see how to use the "MnistClassifier".

Here are the steps to set up the project:

1. Navigate to a directory that you will use, lets call it <working_dir>
2. Run "git clone https://github.com/archer4547/DS_int" in terminal
3. Create a python 3.8 virtual environment. For example, if you hame Anaconda, you can open "Anaconda Prompt" and than:
    1) Run "conda create -n <env_name> python=3.8
    2) Run "conda activate <env_name>
    3) Run "code <working_dir>\DS_int\task_1"
4. Run "pip install -r requirements.txt"
5. Use the classes from the "MnistClassifiers.py" or try the "examples.ipynb"
